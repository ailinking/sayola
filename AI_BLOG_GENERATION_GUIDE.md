## 📋 完整操作步骤

### 第一步：准备AI提示词

使用以下提示词模板来指导AI生成blog文章：

```
请为Sayola葡萄牙语学习平台生成一篇blog文章，要求如下：

主题：[具体主题，例如：葡萄牙语动词变位基础]
目标读者：[初学者/中级/高级]
文章长度：800-1500字

内容要求：
1. 包含实用的葡萄牙语学习技巧
2. 提供具体的例句和翻译
3. 结构清晰，使用标题和列表
4. 包含练习建议或记忆方法
5. 语言简洁易懂，适合中文读者

格式要求：
- 使用Markdown格式
- 包含引言、主要内容、例子、总结
- 每个例句都要有中文翻译
- 使用适当的标题层级（##, ###）

请生成完整的文章内容。
```

### 第二步：生成文章内容

1. 将上述提示词发送给AI（如ChatGPT、Claude等）
2. 检查生成的内容质量
3. 确保葡萄牙语例句准确
4. 验证中文翻译正确性

### 第三步：准备提交信息

收集以下信息用于GitHub Issue提交：

**必需信息：**
- 文章标题（60字符以内）
- 文章分类（Beginner/Intermediate/Advanced/Culture）
- 文章标签（3-8个，逗号分隔）
- 文章摘要（150字以内）
- 完整文章内容（Markdown格式）
- 作者姓名
- SEO标题（60字符以内）
- SEO描述（160字符以内）
- 关键词（5-15个）
- URL路径建议（slug）

### 第四步：创建GitHub Issue

1. 访问：`https://github.com/ailinking/sayola/issues/new/choose`
2. 选择 "Auto Publish Blog Post" 模板
3. 按照模板填写所有信息

**重要：在 `<!-- PASSCODE: 在此处输入暗号 -->` 行下方添加：**
```
woyaofabu
```

### 第五步：提交和验证

1. 点击 "Submit new issue" 提交
2. 等待GitHub Actions自动处理（通常1-2分钟）
3. 检查Issue中的自动回复：
   - ✅ 成功：文章已发布，Issue标记为"published"
   - ❌ 失败：检查错误信息并修正

## 🔧 故障排除

### 常见问题

**1. 暗号验证失败**
- 检查暗号拼写：`woyaofabu`
- 确保暗号在正确位置
- 检查是否有多余空格

**2. 内容解析失败**
- 检查Markdown格式
- 确保所有必填字段已填写
- 验证特殊字符是否正确转义

**3. 文件生成失败**
- 检查文章标题是否包含特殊字符
- 确保slug格式正确（小写字母、数字、连字符）
- 验证JSON格式是否有效
